1. 목표
	-Flowrian을 이용하여 디지털 시스템을 설계하고 구현한다.
	-Top-down 방식의 논리 회로를 직접 설계 및 구현한다. 
	-실생활에 사용 할 수 있는 간단한 일반용 계산기를 구현한다.
2. 주제
	계산기
4. 개발 환경
	-Flowrian
	-Quartus
	-Altera Cyclone FPGA

	-주파수 10KHz 이상에서 동작하도록 구현한다.
